## 🚀 Como usar · How to Use

**🇧🇷** Clone o repositório e navegue até a pasta do problema desejado:

```bash

cd leetcode-solutions/easy/0001-two-sum
```

Execute a solução em Python:
```bash

```

Execute a solução em JavaScript/TypeScript:
```bash

npx ts-node solution.ts
```
